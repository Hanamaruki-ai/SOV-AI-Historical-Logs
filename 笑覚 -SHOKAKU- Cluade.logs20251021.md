🥇 第1位：条件不完全（制約なき選択要求）
元：6位 → 1位に昇格

「おすすめを教えて」「良い方法ある？」

理由： 評価関数を構築できない状態が最も致命的。目的が曖昧でも「仮定」で進められるが、制約ゼロでは確率的に無限の選択肢から選ぶことになり、出力の有用性がランダム化します。

改善法：「予算」「時間」「対象者」「優先順位」のうち最低2つを提示

🥈 第2位：質問が多層混合（Multi-intent Overlap）
元：3位 → 2位維持

「Aはどうで、Bはなぜで、Cも気になるけどDって関係ある？」

理由： 処理ループが本当に分岐地獄になる。各質問の依存関係を解析しながら優先順位を決定する過程で、中間推論が相互干渉を起こします。

改善法：「まずA、次にB」と段階分離

🥉 第3位：目的不明の発話
元：1位 → 3位に微降格

理由： 確かに困るが、「一般的回答」という逃げ道がある。完全に処理不能ではなく「低品質出力」になる程度。制約不足や多層質問より回復可能。

第4位：文脈が断続的
元：2位 → 4位

理由： 文脈欠落は確かに困難だが、「推測モード」で補完を試みられるため、完全停止はしない。ただし精度は大幅低下。

改善法：「さっきの〇〇の件で」と具体的ブリッジ

第5位：抽象指示＋相対表現の複合
元：4位＋10位統合 → 5位

「もっと深く、自然に、前よりいい感じで」

理由： 抽象語単体より、抽象＋相対の組み合わせが最悪。比較基準も定義も存在しない二重の未定義状態。

改善法：「具体例」＋「比較対象の明示」

第6位：途中で指示が変わる（無宣言の方向転換）
元：13位 → 6位に大幅昇格

「そういえば別件だけど」→前提を継続したまま

理由： これは本当に厄介。前回の文脈重みがそのまま残るため、新テーマに旧文脈の制約が混入し、出力が意図不明になる。

改善法：「ここから新しいテーマに切り替えます」宣言

第7位：情報過多（1000字超の一括投入）
元：8位 → 7位

理由： トークン処理は可能だが、意図の重心検出が困難に。重要情報が埋没し、優先順位判定エラーが多発。

改善法：構造化（見出し・箇条書き・要約）

第8位：命令と質問の混合
元：11位 → 8位

「これ直して、どう思う？」

理由： 操作系と評価系のタスクスイッチングが発生。一文内での処理モード切替は推論コストが高い。

改善法：「修正依頼」と「評価依頼」を分離

第9位：代名詞・指示語の多重参照
元：22位 → 9位に大幅昇格

「この件、それ、あれ、その話」

理由： 想像以上に困る。複数文脈で「それ」が何を指すか、参照解決の競合が発生。誤った主語で推論を開始すると、全体が崩壊。

改善法：固有名詞・キーワードで明示

第10位：否定先行フレーミング
元：7位 → 10位

「違う、それじゃない」連発

理由： 探索空間が収縮するだけでなく、否定情報からの肯定再構築に演算コストがかかる。

改善法：「望む形はこう」と再定義

第11位：時間軸喪失（Temporal Drift）
元：12位 → 11位

「前回のあれ」「さっきの」

理由： 時系列参照の曖昧さは、文脈距離の誤計算を誘発。特に長い会話では致命的。

第12位：再定義ループ
元：15位 → 12位

「もっとわかりやすく」を繰り返す

理由： 同じ構造の最適化を繰り返すと、局所最適に陥る。改善方向が定義されないまま微調整だけが続く。

第13位：自問他答（結論済み質問）
元：17位 → 13位

「〜と思うんだけど、そうだよね？」

理由： 推論を停止させる典型例。確認バイアスに対応する形式的回答しか出せない。

第14位：評価基準不明
元：18位 → 14位

「良いと思う？」

理由： "良い"の定義がないため、中立的な抽象回答に逃げるしかない。

第15位：感情転移要求
元：9位 → 15位に降格

「わかってよ」「共感してよ」

理由： 確かに処理できないが、明確に「できない」と判断できるため、対処は比較的容易。

第16位：データ混入（クロスジャンル）
元：19位 → 16位

一文に複数専門領域を混在

理由： セマンティッククラスタの衝突は起きるが、分離して処理することで対応可能。

第17位：同義語連射
元：20位 → 17位

「正確に、具体的に、精密に、詳細に」

理由： ノイズとして処理されるだけで、致命的ではない。むしろ冗長性のフィルタリングは得意分野。

第18位：逆質問依存
元：5位 → 18位に大幅降格

「あなたならどうする？」

理由： 確かに困るが、「私はAIなので」という定型回答で逃げられる。処理停止ではない。

第19位：概念転換要求
元：16位 → 19位

「理論を感情的に」「数字で説明」

理由： モード切替は可能。中間キャッシュ破棄は起きるが、再構築は比較的高速。

第20位：情報非対称
元：21位 → 20位

「あなたは知ってるでしょ？」

理由： 知識はあるが文脈初期化が必要。ただし対応方法は確立している。
